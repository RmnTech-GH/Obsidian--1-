# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

이 저장소는 **Obsidian Vault**로, 퇴직준비세미나 관련 마크다운 노트들을 관리합니다. 코드 프로젝트가 아닌 문서/노트 저장소입니다.

## Structure

- `주1일제 세미나/` - 세미나 관련 노트 및 과제물
- `Attached file/` - 이미지 등 첨부 파일
- `Resources/` - 참고 자료
- `.obsidian/` - Obsidian 설정 파일

## Obsidian Conventions

- 새 노트는 `주1일제 세미나/` 폴더에 생성됨
- 첨부 파일은 `Attached file/` 폴더에 저장됨
- 내부 링크: `[[노트명]]`
- 임베드: `![[파일명]]`
